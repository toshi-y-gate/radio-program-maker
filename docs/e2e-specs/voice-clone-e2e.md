# E2E仕様書: ボイスクローンページ（/voice-clone）

## ゴール
ユーザーがカスタムボイスを作成・管理し、プリセットボイスを確認できること

## テスト項目

| ID | 項目 | 操作手順 | 期待結果 |
|----|------|---------|---------|
| E2E-VC-001 | ページ表示 | /voice-cloneにアクセス | カスタムボイス作成フォーム、プリセットボイス一覧が表示される |
| E2E-VC-002 | プリセットボイス一覧表示 | ページ読み込み完了を待つ | プリセットボイスカード（名前・言語バッジ付き）がグリッド表示される |
| E2E-VC-003 | カスタムボイス作成フォーム入力 | ボイス名入力・音声ファイル選択 | ボイス名が入力欄に表示され、ファイル名が表示される |
| E2E-VC-004 | カスタムボイス作成実行 | ボイス名入力・ファイル選択後に作成ボタンクリック | 登録済みカスタムボイスセクションに新しいボイスが追加される |
| E2E-VC-005 | カスタムボイス削除 | カスタムボイスカードの削除ボタンをクリック | そのボイスがリストから消える |
