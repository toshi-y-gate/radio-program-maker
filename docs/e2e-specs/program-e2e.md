# E2E仕様書: 番組作成ページ（/program）

## ゴール
ユーザーがスクリプトを入力し、音声設定を行い、ラジオ番組を生成・ダウンロードできること

## テスト項目

| ID | 項目 | 操作手順 | 期待結果 |
|----|------|---------|---------|
| E2E-PROG-001 | ページ表示 | ログイン後/programにアクセス | テンプレート選択、スクリプト入力エリア、設定パネルが表示される |
| E2E-PROG-002 | テンプレート選択 | テンプレート（ソロトーク等）をクリック | スクリプト入力欄にテンプレートテキストが挿入される |
| E2E-PROG-003 | 話者自動検出 | 「[話者A] こんにちは」形式のスクリプトを入力 | 話者バッジが表示され、ボイス割り当てカードが出現する |
| E2E-PROG-004 | ボイス割り当て | 検出された話者のドロップダウンからボイスを選択 | 選択したボイスが割り当てられる |
| E2E-PROG-005 | 音声パラメータ調整 | 話速・音量・ピッチのスライダーを変更 | スライダー値が反映される |
| E2E-PROG-006 | 番組生成実行 | スクリプト入力・ボイス割り当て後に生成ボタンクリック | プログレスバーが表示され、生成完了後に音声プレーヤーとダウンロードボタンが表示される |
| E2E-PROG-007 | 生成ボタン無効状態 | スクリプト未入力の状態 | 生成ボタンが無効（disabled）になっている |
