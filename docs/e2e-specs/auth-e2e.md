# E2E仕様書: ログインページ（/login）

## ゴール
ユーザーがアカウント登録・ログインしてアプリにアクセスできること

## テスト項目

| ID | 項目 | 操作手順 | 期待結果 |
|----|------|---------|---------|
| E2E-AUTH-001 | ログインページ表示 | /loginにアクセス | ログインフォーム（メール・パスワード入力欄、ログインボタン）が表示される |
| E2E-AUTH-002 | 新規登録タブ切替 | 「新規登録」タブをクリック | 表示名・メール・パスワード入力欄と登録ボタンが表示される |
| E2E-AUTH-003 | 新規登録成功 | 表示名・メール・パスワードを入力して登録ボタンクリック | /programに遷移する |
| E2E-AUTH-004 | ログイン成功 | 登録済みのメール・パスワードでログイン | /programに遷移する |
| E2E-AUTH-005 | バリデーションエラー表示 | 無効なメール形式でログイン試行 | エラーメッセージが表示される |
| E2E-AUTH-006 | パスワード強度表示 | 新規登録タブでパスワードを入力 | パスワード強度インジケーターが表示・更新される |
